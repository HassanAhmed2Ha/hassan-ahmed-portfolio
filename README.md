# Hassan Zaki – Modern Personal Portfolio Template

A clean, modern, and highly scalable personal portfolio template built for students, researchers, and early-career professionals. 

**Live Portfolio:** [https://hassan-ahmed-portfolio.vercel.app](https://hassan-ahmed-portfolio.vercel.app)

---

## 🚀 The Philosophy: Content-First Architecture
Unlike traditional portfolio templates that require digging into JSX/TSX files to change a single text, this project uses a **Content-First Architecture**.

- **Single Source of Truth:** All your personal data, experiences, projects, and certifications are managed in one file: `src/data.ts`.
- **Developer Experience:** You only need to edit `src/data.ts` to update your entire website. No coding required for content updates.
- **Modern Stack:** Built with [Next.js 15](https://nextjs.org/), TypeScript, and Tailwind CSS for maximum performance and responsiveness.

---

## 🛠 Features
- **Clean-Tech Aesthetic:** Grid-based layouts, glassmorphism design, and smooth user experience.
- **Fully Responsive:** Optimized for mobile, tablet, and desktop.
- **Easy Customization:** Edit `src/data.ts` and see your changes reflect instantly.
- **Deployment Ready:** Configured for seamless deployment on Vercel.

---

## ⚙️ Quick Setup

### 1. Fork & Clone
Fork this repository to your GitHub account and clone it to your local machine.

### 2. Configure Your Content
Open `src/data.ts`. This file contains the `contentEn` object.
- Update your **Hero** section (title, role, description).
- Fill in your **About** section, **Experience**, and **Certifications**.
- Add your **Projects** and **Publications**.

### 3. Deploy
1. Push your changes to your `main` branch.
2. Import the repository to [Vercel](https://vercel.com/).
3. Vercel will automatically detect the Next.js configuration. Click **Deploy**!
